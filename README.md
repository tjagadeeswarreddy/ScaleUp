# ScaleUP Agency Website

A professional, high-impact landing page for ScaleUP, a digital marketing agency specialized in brand growth, reels, and performance-driven marketing. This website is a digital representation of the agency's official presentation brochure.

## 🛠️ Tech Stack
- **React 18**
- **TypeScript**
- **Vite**
- **Framer Motion**
- **Lucide React**
- **Vanilla CSS**

## 🚀 Key Features
- **Reel Showcase:** Interactive mock-phone video player.
- **Strategic Workflow:** Visual 6-step business process.
- **Service Mastery:** Comprehensive grid for SEO, Content Creation, and Branding.
- **Lead Generation:** Integrated Floating WhatsApp and Contact Form.
- **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop.

## 🏁 Getting Started
1. `npm install`
2. `npm run dev`
